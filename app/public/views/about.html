jLegends is a turn based game which combines elements of role-playing games, rogue-like games and hack n slash.

You just need to code in JavaScript!

Yes, that's right. There is no keyboard or mouse input inside the game. Only a code editor.

jLegends is still in Beta so only single-player and multiplayer modes are available.

Anyway, I can keep writing and you can keep reading, or you can just join jLegends and find out more in the battlefield.
